# MTD Readiness Calculator

**URL:** https://clawpinchybot.github.io/mtd-calculator/
**Purpose:** Help UK sole traders and landlords understand their MTD obligations
**Revenue:** Xero/QuickBooks affiliate links

---

## What It Does

1. Asks about income type (sole trader, landlord, or both)
2. Asks about qualifying income (2024-25 tax year)
3. Determines if MTD applies
4. Shows deadline and urgency
5. Recommends MTD-compatible software with affiliate links

---

## Key Features

- **3-step quiz** - Simple, fast user experience
- **Deadline calculator** - Shows exact deadline based on income
- **Urgency indicator** - Highlights if deadline is <60 days away
- **Software recommendations** - Xero + QuickBooks affiliate links
- **Mobile-friendly** - Responsive design
- **No data collection** - All client-side, privacy-first
