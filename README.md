## What This Tool Does

A simple calculator that:
1. Asks about income sources
2. Calculates if/when they need to comply
3. Shows clear deadline and requirements
4. Recommends MTD-compliant software
5. Earns affiliate commissions on signups

**Why it works:**
- Free = high traffic potential
- Answers urgent question
- Natural upsell to software
- No backend needed (pure HTML/JS)
- Can rank for "MTD calculator" queries

### Step 3: Deploy

**Option A: GitHub Pages (FREE)**
```bash

git remote add origin https://github.com/YOUR_USERNAME/mtd-calculator.git

```
